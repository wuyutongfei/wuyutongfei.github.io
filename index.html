处理样式差异的常用手段

处理样式差异的常用的三种方式

第一种 reset.css 或者 normalize.css

★ Normalize 相对「平和」，注重通用的方案，重置掉该重置的样式，保留有用的 user agent 样式，同时进行一些 bug 的修复，这点是 reset 所缺乏的。

★ Reset 相对「暴力」，不管你有没有用，统统重置成一样的效果，且影响的范围很大，讲求跨浏览器的一致性。

mvc和mvvm区别

mvc

MVC是一种架构模式，M表示Model，V表示视图View，C表示控制器Controller
Model负责存储、定义、操作数据；
View用来展示给用户，并且和用户进行交互；
Controller是Model和View的协调者，Controller把Model中的数据拿过来给View使用。

织梦系统 遇到得各种bug

问题1：把数据保存到数据库附加表 dede_addonarticle 时出错，请把相关信息提交给DedeCms官方

解决方案：进入织梦后台，“系统” - “系统设置” - “SQL命令行工具”
运行SQL命令行：alter table dede_addonarticle drop primary key
问题就解决了

问题2：织梦后台添加一条数据之后 前台自动生成2条

解决方案：清空dede_addonarticle 这个表（不会对网站造成任何影响）
重新生产一下，问题就解决了
