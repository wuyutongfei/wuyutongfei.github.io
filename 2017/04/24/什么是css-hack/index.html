css hack分类

由于不同厂商的浏览器或某浏览器的不同版本对css的支持、解析不一样,所以显示效果会出现微小差异。
借助解析差异和优先级差异，分别为不同浏览器写样式
注：一般是将识别能力强的浏览器的CSS写在后面
